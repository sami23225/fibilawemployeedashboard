Set Up Airtable Integration
To connect your Airtable bases (Sales, Refunds, ESL, ECAS), you'll need an Airtable Personal Access Token.

How to get your token:
1. Go to airtable.com/create/tokens
2. Click "Create token"
3. Add scopes: data.records:read and schema.bases:read
4. Add access: "All current and future bases in all current and future workspaces"
5. Copy the token (starts with pat...) and paste it below

Set Up Calendly Integration
To pull upcoming consultations from your Calendly account, paste a Personal Access Token.

How to get your token:
1. Go to calendly.com/integrations/api_webhooks
2. Click "Generate New Token" under Personal Access Tokens
3. Give it a name like "CEO Dashboard"
4. Copy the token and paste it below

The token is stored only in your browser. Never commit it to GitHub or share it.
